# Infografía Interactiva Web

## 📌 Descripción
Este proyecto presenta una infografía interactiva basada en la síntesis de un paper científico, transformando información académica en un formato visual claro, dinámico y accesible para entornos digitales educativos.

## 🚀 Tecnologías utilizadas
- HTML5
- CSS3
- GitHub Pages
- Inteligencia Artificial (Z.ai)

## 💡 Uso de IA
Se utilizó Z.ai para la generación del código frontend (estructura HTML y estilos CSS) de la infografía web.
