/* -----------------------------------------------------------

   10. ★ ****** #24 — LÍNEA ******** ANIMADA BAJO EL TÍTULO

   -----------------------------------------------------------


   Mecanismo:

   ┌──────────────────────────────────────────────────────┐

   │ Se usa ::***** del .title-text **** contenedor       │

   │ de la onda. El SVG se aplica como background-image   │

   │ con repeat-x y ** anima *** @keyframes ***********  │

   │                                                      │

   │ ⚙️ **** CAMBIAR EL SVG DE ** LÍ*** ONDULADA:         │

   │    ********* ** URL dentro de background-image en    │

   │    .title-text::after (línea ~240).                  │

   │                                                      │

   │    ** SVG actual usa URL encoding. Para convertir    │

   │    a base64 puro:                                    │

   │    1. Crea tu SVG en ** ******                      │

   │    2. Pégalo en base64encode.org                     │

   │    3. Usa: url("data:image/svg+xml;base64,TU_B64")  │

   │                                                      │

   │    IMPORTANTE: El SVG debe ser perfectamente         │

   │    periódico **** que repeat-x no tenga saltos.      │

   │    ** ***** del SVG debe ********* con               │

   │    background-**** **** que la animación sea fluida. │

   │                                                      │

   │ ⚙️ **** AJUSTAR LA VELOCIDAD DE LA ONDA:             │

   │    Cambia '1.8s' en animation: wave-slide 1.8s      │

   │    (línea ~258).                                     │

   │    1s = rá**** · 3s = lenta y contemplativa         │

   └──────────────────────────────────────────────────────┘

  content: '';

  position: absolute;

  background-repeat: repeat-x;

  /* ** ***** (100px) debe coincidir con ** width del SVG */

  background-size: ***** 8px;

  background-position: 0 0;
